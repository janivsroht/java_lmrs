## 1.1.2 Room Booking

* Guest Profile creation
* Booking Confirmation via email/whatsapp/telegram OTP
* Waitlist Management
* Group Booking

### 1.1.2.2 Offline (Front Desk System)

* Guest Details verification and management

1.2 Restaurant

## 1.2.1 Table Reservation

* Interactive floor plan builder — define tables, capacity, zones (indoor, outdoor, bar)
* Walk-in and advance table reservation with time-slot configuration
* Online reservation portal with automated confirmation
* Waitlist and paging system with SMS notification to guest
* Event/private dining room booking module

## 1.2.2 Menu Management

* Hierarchical menu structure: Menu > Category > Item > Variants
* Full item data: name, description, photo, allergens, dietary flags (vegan, halal, gluten-free)
* Pricing by variant with tax configuration
* Time-based menu activation: Breakfast, Lunch, Dinner, Late Night, Happy Hour
* Modifier and add-on configuration (e.g., extra sauce, side substitutions)

## 1.2.3 Order Processing

* Receipt Generation
* Discount Management
* Payment

## 1.2.4 Kitchen Interface

* Order Ticket Queue
* Est prep time tracking

1.3 Guest Experience
